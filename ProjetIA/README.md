# Projet IA — Adversarial Attacks on Deep Learning Models

Study of adversarial attacks (FGSM and related methods) against image
classification models: generating adversarial examples, evaluating model
robustness, and correcting/iterating on attack implementations.

- `ProjetIA_Adversarial_Part1.ipynb`, `Projet_Deep_Adversarial2.ipynb`,
  `Deep_Adv_Adversarial2_FSGM.ipynb`, `copie_de_adversarial3.ipynb` — attack
  implementations and experiments
- `ADV2_Correction.ipynb` — corrected/reference version
- `Projet IA.pptx` / `Projet IA (1).pptx` — project presentation slides
