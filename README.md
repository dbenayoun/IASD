# IASD — Master in Artificial Intelligence, Paris Dauphine University

Coursework and projects from my Master's in Artificial Intelligence at Paris
Dauphine University (Executive IASD program): machine learning, deep
learning, NLP, computer vision, graph algorithms, and distributed data
processing.

## Projects

| Folder | Topic |
|---|---|
| [`DeepLearning/`](DeepLearning) | Deep learning notebooks: CNNs (MNIST, CIFAR-10), autoencoders, text classification (DistilBERT, TinyLlama fine-tuning), text convolution |
| [`Flight/`](Flight) | Flight delay prediction using weather data joined with flight records — Random Forest classifier, Spark/Scala pipeline |
| [`GenerativeAI_RAG/`](GenerativeAI_RAG) | Retrieval-Augmented Generation (RAG) backend with Flask, OpenAI API, and ChromaDB for customer service response generation |
| [`Graph - Connected Component/`](Graph%20-%20Connected%20Component) | Distributed connected-component algorithm (CCF paper) implemented with Spark DataFrames and RDDs |
| [`Optimiasation - SVM/`](Optimiasation%20-%20SVM) | Optimal separating hyperplane via Mixed Integer Linear Programming (no-loss, hinge loss, ramp loss models) |
| [`ProjetIA/`](ProjetIA) | Adversarial attacks (FGSM) on deep learning image classifiers |
| [`Reco system/`](Reco%20system) | Recommender system on the MovieLens dataset |
| [`SQL/`](SQL) | Delta Lake exercises (Spark SQL, use-case notebooks) |
| [`Misc/`](Misc) | Other standalone coursework notebooks (data quality, ticket classification, etc.) |

Each folder contains its own README with more detail where relevant.
